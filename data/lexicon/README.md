# NRC Emotion Lexicon (EmoLex) — vendored copy

File: `nrc_emotion_lexicon.json`

Source: NRC Word-Emotion Association Lexicon (EmoLex), created by Saif M. Mohammad
and Peter D. Turney, National Research Council Canada.

Reference:
Mohammad, S. M., & Turney, P. D. (2013). Crowdsourcing a word-emotion association
lexicon. Computational Intelligence, 29(3), 436-465.

This lexicon maps English words to one or more of: anger, anticipation, disgust,
fear, joy, sadness, surprise, trust, positive, negative.

License / usage note: the lexicon is free to use for research purposes. Any
commercial use requires checking the current license terms from the NRC. This
copy is included only so the lexicon-based baseline in this repository can run
fully offline, without depending on a third-party package at run time. It is
data, not original code, and is not covered by this repository's own LICENSE
file.
